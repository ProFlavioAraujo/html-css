Pseudo-classes

As Pseudo-classes estão relacionadas a um id ou a uma classe, e são representadas pelo sinal : (dois pontos). A pseudo-classe define o estado atual de um determinado elemento. Por exemplo, se este elemento está: ativo, selecionado vazio, habilitado, entre outros estados.
